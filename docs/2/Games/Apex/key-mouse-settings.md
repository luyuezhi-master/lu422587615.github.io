# Apex 键鼠设置

- 鼠标
    - 左键 开火
    - 右键 按住开镜
    - 中键 标记
    - 滚轮上 前进
    - 滚轮下 跳跃
- 键盘
    - 移动
        - W/A/S/D 移动
        - SPACE 跳跃
        - SHIFT ALT 下蹲
    - 交互
        - F 交互
        - T 地图
        - TAB 背包
    - 武器
        - C/V/X 武器1 武器2 收起武器
        - R 再装填
    - 使用
        - E/Q 小技能 大技能
        - 1/2/3/4/5 小药大药小电大电凤凰治疗包
        - G 手雷
        - H 求生物品
    - 其他
        - Y 按住说话
        - CAPS 标记敌人
        - Z 切换缩放
        - \` 谢谢
        - ENTER 发送信息
